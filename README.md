# ShopperAssistant / 超市助手

一个给超市日常工作使用的离线 Android 原生工具。应用提供条码扫码生成和保质期检查两类功能，界面以简体中文为主，使用 Jetpack Compose 和 Material 3，支持系统深色/浅色模式。

## 功能

- 扫描常见一维条码：EAN-13、EAN-8、UPC-A、UPC-E、Code39、Code128。
- 将扫码或手动输入的内容生成 Code39 条形码 PNG。
- 条形码下方显示对应字符，便于人工核对。
- 支持保存和分享生成的 PNG。
- 支持保存、恢复和删除条码历史记录。
- 通过生产日期和保质期天数检查商品是否过期。
- 保质期按“生产日期当天算第 1 天”计算；到期日当天判定为已过期。
- 完全离线运行，不接入商品数据库、账号系统或网络接口。
